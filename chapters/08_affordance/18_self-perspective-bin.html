Self-perspectival recycle bins

Not only flat images, 3-d shapes too interact with perspectival change. The designer of this set of recycling bins skewed the parallelepipedal shape of the containers. The result is a discomforting effect when passing by. But definitely attention-grabbing, which is what one wishes for recycling.
